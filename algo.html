Алгоритм вычесляющие исходы с 1+ профессиями

Чтобы дальше выявлять параметры можно использовать хороший метод - найти те расклады где результат будет n>1 профессий. И сравнить интуитивно эти профессии - там будут обнаружены различия. Алгоритм сделан таким образом что мы сравниваем профессии комбинациями(парами). Снизу выделены теперь уже все возможные комбинации при определенных ходах(если input range от 0 до 100, то L - это значение [0,40] а R - [60,100])
